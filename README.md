## 搬瓦工 VPS 是什么？先说清楚几个基本概念

**搬瓦工**（英文名 BandwagonHost）是加拿大 IT7 Networks 旗下的 VPS 服务商，成立以来专注高性能 KVM 虚拟化 VPS，对国内用户做了大量线路优化。之所以叫"搬瓦工"，单纯是因为 BandwagonHost 的拼音近似于 bān wǎ gōng，叫顺了就传开了。

VPS 是 Virtual Private Server 的缩写，本质上是一台在云端的虚拟服务器，有独立 IP、完整系统权限、可以装任何你想装的软件。搬瓦工的 VPS 全部基于 KVM 虚拟化，这点很重要——KVM 比早期的 OpenVZ 隔离性好得多，性能也更稳。

搬瓦工 VPS 对国内用户最大的吸引力在两件事：**CN2 GIA 线路**和**可以用支付宝付款**。前者是技术层面的核心优势，后者是付款便利。

## 搬瓦工的线路类型到底有什么区别

这是买搬瓦工 VPS 最容易踩坑的地方。同样是"美国机房"，线路不同，晚高峰的体感差距可以是天壤之别。

搬瓦工目前主要覆盖四种线路质量：

**普通 KVM 线路（入门款）**
走的是普通国际带宽，没有针对国内做特别优化。适合建站学习、跑一些对延迟不敏感的应用。晚高峰可能会有卡顿，但价格最低，年付 $49.99 起。

**CN2 GT**
比普通线路好一些，是中国电信 CN2 系列里的入门级。以前比较有优势，这两年拥堵情况有所加剧，但仍然比裸 BGP 稳。

**CN2 GIA-E（推荐主力）**
这是搬瓦工性价比最高的一档。CN2 GIA 是中国电信最高等级线路，采购成本极高，但搬瓦工把它打进了 $49.99/季 的价位。走三网优化：电信 CN2 GIA + 联通 9929 + 移动 CMIN2，晚高峰稳定性明显好于入门款，支持 12 个以上机房自由切换。

**香港/日本 CN2 GIA（顶级）**
物理距离近，延迟可以压到极低。香港机房到广东部分地区延迟能达到 30ms 以内。代价是价格贵很多，最低 $89.99/月，不是普通用户的首选。

简单讲：**预算有限就选 KVM 入门套餐；想要稳定速度快选 CN2 GIA-E；钱不是问题或者对延迟有极致要求，选香港/日本 CN2 GIA。**
